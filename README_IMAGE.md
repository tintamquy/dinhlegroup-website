# Hướng dẫn thêm ảnh Social Preview

## Yêu cầu ảnh:
- **Tên file:** `og-image.png`
- **Kích thước:** 1200 x 630 pixels (tỷ lệ 1.91:1)
- **Định dạng:** PNG hoặc JPG
- **Nội dung:** Logo Dinh Le Group với nền xanh dương và text "Since 2001"

## Cách thêm:
1. Đặt ảnh logo của bạn vào thư mục gốc của website
2. Đổi tên file thành `og-image.png`
3. Đảm bảo kích thước là 1200x630px (có thể resize bằng Photoshop, Canva, hoặc công cụ online)

## Kiểm tra:
Sau khi deploy, bạn có thể test social preview tại:
- Facebook: https://developers.facebook.com/tools/debug/
- Twitter: https://cards-dev.twitter.com/validator
- LinkedIn: https://www.linkedin.com/post-inspector/

## Lưu ý:
- Ảnh phải có kích thước đúng để hiển thị đẹp trên mạng xã hội
- Nên sử dụng ảnh có độ phân giải cao
- File size nên dưới 1MB để load nhanh
